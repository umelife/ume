# Placeholder Images for Homepage

This directory contains placeholder images used in the homepage components. Replace these with your actual images to customize the appearance.

## Required Images

### Hero Section
- **File:** `hero-city.jpg`
- **Recommended size:** 1920x1080px or higher
- **Aspect ratio:** 16:9
- **Description:** City skyline or campus background image for the hero section
- **Current placeholder:** Gray gradient with city silhouette

### Feature Slider
- **File:** `feature-chat.jpg`
  - Size: 1920x1080px
  - Description: Background for "REAL-TIME CHAT" slide (ocean/sunset scene works well)

- **File:** `feature-secure.jpg`
  - Size: 1920x1080px
  - Description: Background for "SECURE PAYMENTS" slide

- **File:** `feature-local.jpg`
  - Size: 1920x1080px
  - Description: Background for "LOCAL COMMUNITY" slide

## Image Guidelines

- **Format:** JPG or PNG
- **Optimization:** Compress images to keep file sizes under 500KB each
- **Quality:** Use high-quality images that match your brand aesthetic
- **Rights:** Ensure you have the rights to use all images

## Using Next.js Image Optimization

All images are loaded through Next.js `<Image>` component which provides:
- Automatic optimization
- Lazy loading
- Responsive sizing
- Modern format conversion (WebP)

No additional configuration needed - just drop your images in this folder!
